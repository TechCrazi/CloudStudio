CloudStudio API

Unified API access for overview, billing, cloud metrics, security, and platform integrations.

Authentication

Public API endpoints require x-api-key. Create and manage keys in CloudStudio under Admin â†’ API keys.

Admin UI: sign in, then open the Admin menu in CloudStudio.

Public Endpoints

Base path: /api/public/*

curl -H "x-api-key: YOUR_KEY" https://example.com/api/public/overview
curl -H "x-api-key: YOUR_KEY" https://example.com/api/public/billing
curl -H "x-api-key: YOUR_KEY" "https://example.com/api/public/cloud-metrics?provider=azure"
curl -H "x-api-key: YOUR_KEY" "https://example.com/api/public/cloud-metrics?provider=aws"
curl -H "x-api-key: YOUR_KEY" "https://example.com/api/public/cloud-metrics?provider=rackspace"
curl -H "x-api-key: YOUR_KEY" "https://example.com/api/public/cloud-metrics/resources?provider=aws&limit=500"
curl -H "x-api-key: YOUR_KEY" https://example.com/api/public/security

Core Private API (App/Auth Required)

  • GET/POST/PUT/DELETE /api/vendors
  • GET /api/billing, GET /api/billing/accounts, GET /api/billing/export/csv
  • POST /api/billing/backfill, POST /api/billing/backfill/ensure, GET /api/billing/backfill/status
  • GET /api/tags, POST /api/tags/resolve, POST /api/tags/sync/aws, PUT /api/tags
  • POST /api/platform/pricing/compare
  • GET /api/platform/overview, /cloud-metrics, /cloud-metrics/providers, /live, /security
